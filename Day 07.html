ADDING PRESSURE IN WORDS

The words taught uptill now does not express anger or surety. For that we:
Rule 1 : Add ل with zabr on it as first letter.
Rule 2 : Add ن in the end with zabr and tajdeed on it.

ORDERS

How to give orders (i.e. do this and do that) in arabic. For that we :


Rule 1 : Remove the First Letter

Rule 2 : Replace Zer/Zabr/Pesh from the end with sakin or if there is "ن" in the end, then remove it.

Rule 3 : If first letter consists of sakin then put "ا" in the starting.

Note: Order is only in Present / Future tense. Words from Past tense cannot be used. Also the word can be a singular, plural.


FOR EXAMPLE
Verbs Rule 1 Rule 2 Rule 3 Meaning
Go [alone] and beat [him/her/them/it].
Go all and beat [him/her/them/it].
Go [alone] and earn.
Since there is no sakin in the starting hence 3rd rule will not be applied. You [alone] be in tension.
